Overview

simple_docker is an Eiffel library for programmatic Docker container management. It provides a clean, type-safe API for building, running, and managing Docker containers.

Container Lifecycle

Create, start, stop, pause, restart, and remove containers with full state tracking.

Image Management

List, pull, inspect, and remove Docker images from local and remote registries.

Fluent Builder API

Configure containers using intuitive chained method calls.

Design by Contract

Full DBC support with preconditions, postconditions, and invariants.

Installation

Requirements

  • EiffelStudio 25.02 or later
  • Docker Desktop running on Windows
  • Dependencies: simple_ipc, simple_json, simple_file, simple_logger

Setup

# Clone the repository
git clone https://github.com/simple-eiffel/simple_docker.git

# Set environment variable
export SIMPLE_DOCKER=/path/to/simple_docker

ECF Configuration

<library name="simple_docker" location="$SIMPLE_DOCKER\simple_docker.ecf"/>

Quick Start

local
    client: DOCKER_CLIENT
    spec: CONTAINER_SPEC
    container: detachable DOCKER_CONTAINER
do
    -- Create client (connects via named pipe)
    create client.make

    -- Check Docker is running
    if client.ping then
        print ("Docker daemon is responsive%N")
    end

    -- Create container specification
    create spec.make ("alpine:latest")
    spec.set_name ("my-container")
        .set_cmd (<<"echo", "Hello!">>)
        .add_env ("FOO", "bar").do_nothing

    -- Create and start container
    container := client.run_container (spec)

    if attached container as c then
        print ("Created: " + c.short_id + "%N")

        -- Wait for completion
        client.wait_container (c.id).do_nothing

        -- Get logs
        if attached client.container_logs (c.id, True, True, 100) as logs then
            print (logs)
        end

        -- Cleanup
        client.remove_container (c.id, True).do_nothing
    end
end

API Reference

DOCKER_CLIENT

Main facade for all Docker operations.

FeatureDescription
pingCheck if Docker daemon is responsive
versionGet Docker version information
infoGet Docker system information
list_containers (all)List containers (optionally all)
create_container (spec)Create container from spec
start_container (id)Start a container
stop_container (id, timeout)Stop with timeout
remove_container (id, force)Remove container
list_imagesList all images
pull_image (name)Pull image from registry

CONTAINER_SPEC

Fluent builder for container configuration.

FeatureDescription
set_name (name)Set container name
add_port (container, host)Map TCP port
add_env (key, value)Add environment variable
add_volume (host, container)Mount volume
set_memory_limit (bytes)Set memory limit
set_restart_policy (policy)Set restart policy
to_jsonExport as JSON for Docker API

DOCKER_CONTAINER

Container representation with state queries.

FeatureDescription
id / short_idContainer ID
nameContainer name
is_runningCheck if running
is_exitedCheck if exited
can_startCan container be started?
exit_codeExit code (if exited)

DOCKER_IMAGE

Image representation with metadata.

FeatureDescription
id / short_idImage ID
repo_tagsList of tags
size / size_mbImage size
has_tag (tag)Check for tag
matches (ref)Match by name or ID

Examples

Web Server Container

create spec.make ("nginx:alpine")
spec.set_name ("web-server")
    .add_port (80, 8080)
    .add_volume ("/var/www", "/usr/share/nginx/html")
    .set_restart_policy ("unless-stopped").do_nothing

container := client.run_container (spec)

Error Handling

container := client.create_container (spec)

if client.has_error then
    if attached client.last_error as err then
        if err.is_not_found then
            -- Image not found, pull it
            client.pull_image (spec.image).do_nothing
        elseif err.is_conflict then
            -- Container name exists
        elseif err.is_retryable then
            -- Network error, can retry
        end
    end
end

Listing and Filtering

-- List running containers
across client.list_containers (False) as c loop
    if c.is_running then
        print (c.name + " is running%N")
    end
end

-- Find image by tag
across client.list_images as img loop
    if img.has_tag ("alpine") then
        print ("Found alpine: " + img.short_id + "%N")
    end
end
